BU Dev studio

Проект по разработке программного обсепечения

от BIONIC University

BIONIC University в рамках развития концепции прктического обучения запустил собственный проект по разработке програмного обеспечения - BU Dev Studio. Участники проекта работают над продуктом в команде под руководством опытных тим-лидеров, таким образом, изучая на практике весь цикл разработки продукта.Участие в проектах предпологает наличие у кандидата высокого уровня подготовки и будет доступно только выпускникам BIONIC School. Срок сразработки бизнес-решения: 15-15 недель. Команды работают по методологии управления проектами Scrum или Iterative Waterfall. Это вовлекаеткоманду в активное взаимодействие с заказчиком: демо,релизы, обратная связь.Разработка программного продукта в BU Dev Studio дает практический опыт работы над реальными бизнес-задачами.

Интернатура в BU Dev Studio по направлению Java является завершающим этапом подготовки Java Junior Developer в BIONIC University, и дает выпускникам опыт работы на реальном проекте. Участие в интернатуре требует вовлеченности в среднем 15-20 часов в неделю, где соотношение теоретической подготовки к практической разработке составляет 1:3 соответственно.   Команды разрабатывают продукт для реальных заказчиков под руководством тим-лидов. Весь период разработки составляет 12-15 недель и разбит на 6-8 итераций, каждая из которых завершается демо. Разработка проекта выполняется по методологии Scrum или Iterative Waterfall. Независимо от выбранной методологии разработки, тим-лиды (middle/senior Java разработчики из аутсорсинговых и продуктовых компаний) проводят code review, помогают проводить estimate задач и планировать backlog. Проведение демо и milestone релизы позволяют собрать фидбек от стейкхолдеров, что способствует получению более качественного продукта на выходе. Выпускник Java интернатуры в BU Dev Studio – это Java Junior Developer с навыками и знаниями, соответствующими современным требованиям IT-компаний:

  • 3-4 месяца опыта работы на проекте,
  • глубокие практические знания современных Java технологий (Spring MVC,  REST API и т.д.),
  • умение работать с продуктами Atlassian (Jira, Bitbucket, Confluence),
  • опыт командной разработки по Agile методологиям,
  • опыт взаимодействия с основными стейкхолдерами проекта (product owner/бизнес-аналитик, потенциальные пользователи и др.),
  • реальный проект в портфолио.

Интернатура в BU Dev Studio по направлению BA+QA является уникальным и комплексным этапом подготовки Junior QA/BA в BIONIC University, и дает выпускникам возможность не только получить глубокую базу теоретических знаний в сфере Manual QA и Business Analysis, но и сразу же отточить их на реальном проекте. Участие требует значительной временной вовлеченности, мотивации развиваться в данной сфере, отличных коммуникационных скиллов и хорошего уровеня владения английским языком. Студенты BA+QA интернатуры в BIONIC University проходят период обширной теоретической подготовки по направлениям Manual QA и Business Analysis с последующим включением в команды интернов-разработчиков, задействованных в реализации проектов. Команды разрабатывают продукт для реальных заказчиков в течении 12-15 недель. BA+QA интерны вливаются в команду, и становятся очень важной и неотъемлемой её частью. Они на практике выполняют все функции тестировщика и бизнес-аналитика в ходе реализации проекта, получают комплексное понятие о командной работе, медиации между заказчиком и разработчиками, ведут всю необходимую документацию по проекту, собирают и передают фидбек, приобретают практический опыт manual testing и работы с Jira в системе гибкой итеративной разработки (разработка проектов выполняется по методологии Scrum или Iterative Waterfall). Опытные тим-лиды (middle/senior QA/BA из аутсорсинговых и продуктовых компаний) менторят и фасилитируют процесс реализации закреплённого за интернами функционала, тем самым способствуя получению всей командой более качественного продукта на выходе.
